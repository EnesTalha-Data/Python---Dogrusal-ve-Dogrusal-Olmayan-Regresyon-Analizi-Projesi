# Python---Dogrusal-ve-Dogrusal-Olmayan-Regresyon-Analizi-Projesi
## Verinin Hikayesi
Bu proje Kaggle üzerinden alınmış "Chicago House Price" veri seti ile yapılmıştır. Amaç Chicago'daki ev fiyatlarını hangi değişkenlerin en çok etkilediğini ve en uygun regresyon modelini kurmaya yöneliktir. Veri setinde yer alan değişkenler şunlardır;
1-) Price - Yanıt değişkenimiz. Ev fiyatlarını gösterir.
